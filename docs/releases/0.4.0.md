# Dataset release 0.4.0

- Added atomic predicates, composed policies, catalog-owned coverage, and per-rule-target
  retirement as four normalized SQLite tables.
- Migrated 15 representative methods without claiming complete coverage for partially audited
  assumptions.
- Made retired rule targets executable only after predicate-policy validation while preserving
  the existing recommendation result and trace contract.
- Added the same predicate records to Map details, method pages, and SiteData 2.0.0.
- Added a deterministic report of remaining free-text-only method condition fields.
