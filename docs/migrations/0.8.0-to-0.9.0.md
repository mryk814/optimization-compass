# Dataset migration 0.8.0 → 0.9.0

The old `prerequisite`, `next`, `related`, and `contrast` edge vocabulary is replaced by the
typed learning relations defined in Issue #66. Consumers should read `learning-graph.json` or the
normalized `learning_edges` and `terminology_aliases` tables. Alias collisions are valid only when
every candidate carries an explicit disambiguation note; deprecated terms remain searchable but
are marked separately from current labels and synonyms.
