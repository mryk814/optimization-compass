# Dataset migration 0.9.0 → 0.10.0

No table was added or removed. `method_visualization_profiles.family` now accepts
`feasible_region` and `pareto_front`; two canonical `demo_scenarios` and their profiles were added.

`INSTANCE_CONSTRAINED_DISK_2D` is corrected from an inconsistent outside-disk inequality to the
inside-disk inequality `(x-1)^2+(y-1)^2 <= 1`, which agrees with its exact active-boundary optimum.
Consumers should use the generated problem catalog rather than retaining the earlier expression.
